# DPP多样性算法实现


## 文件说明

- demo文件夹下面是Java版本的实现demo

- other文件夹下面是C++版本和Python版本的实现demo

- src文件夹下是一个SpringBoot工程，将DPP算法工程化，对外提供REST服务。
